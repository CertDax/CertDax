# Security Policy

If you discover a security vulnerability or incident, please notify us as soon as possible. We appreciate responsible disclosure and your help to keep CertDax safe for everyone.

## Reporting a Vulnerability

- Please contact us via email at **info@squadrasec.com** OR by creating a security issue on this GitHub repository.
- Include as much detail as possible to help us understand and reproduce the issue.

## Response Policy

- Response times: **We respond to security reports exclusively on weekends (Saturday and Sunday)**. Please allow for delays if you report an issue during the week.
- We will make every effort to resolve security issues as quickly as possible once we are aware of them.

## Responsible Disclosure Guidelines

- Please report potential security issues as soon as possible after discovery.
- Please allow us a reasonable amount of time to resolve the issue before public disclosure or disclosure to a third-party.
- If appropriate, we may publicly disclose the issue before resolving it.
- Make a good faith effort to avoid privacy violations, destruction of data, and interruption or degradation of our service.
- Only interact with your own accounts or those for which you have explicit permission.

Thank you for helping us keep CertDax secure!
